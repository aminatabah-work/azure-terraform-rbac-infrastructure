# Azure Terraform RBAC Infrastructure

This project demonstrates how to provision secure Azure infrastructure using Terraform with a focus on role-based access control (RBAC), reproducibility, and cloud infrastructure best practices.

## Project Goals

- Provision Azure infrastructure using Terraform
- Apply secure-by-default infrastructure practices
- Demonstrate RBAC concepts in Azure
- Build a simple, professional cloud engineering portfolio project

## Planned Resources

- Resource Group
- Virtual Network
- Subnet
- Storage Account
- Virtual Machine
- RBAC role assignments

## Tools Used

- Terraform
- Microsoft Azure
- GitHub
- Infrastructure as Code (IaC)

## Why I Built This

I created this project to strengthen my Azure cloud engineering skills and demonstrate hands-on experience with Terraform, infrastructure automation, and secure access design.
